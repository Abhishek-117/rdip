In the First week of the RDIP conduted by IIIT Hyderabad, I was able to learn how to create Webpages using HTML and CSS.

			 HTML - HyperText Markup Language :-
			 	1)  HTML stands for Hyper Text Markup Language.
				2)  HTML is the standard markup language for creating Web pages.
		   		3)  HTML describes the structure of a Web page.
				4)  HTML consists of a series of elements.
				5)  HTML elements tell the browser how to display the content.
				6)  HTML elements label pieces of content such as "heading", "paragraph", "link", etc.
				7)  An HTML element is defined by a start tag, some content, and an end tag.
				8)  A HTML design consist of "html" element which encloses "head" and "body" elements.
				9)  The current version of the HTML is HTML5.
				10) HTML is denoted by "!DOCTYPE html" inside a tag at the starting of the HTML code, 
						which specifies that it is a HTML5 document.

			 CSS- Cascading Style Sheets :-
			 	1)  CSS is a language that describes the style of an HTML document.
				2)  CSS describes how HTML elements should be displayed.
				3)  CSS is a simple mechanism for adding style (e.g., fonts, colors, spacing) to Web documents.
				4)  CSS consists huge library of fonts, color and other styling properties from which we can alter our 
						HTML document to seem Attractive.
				5)  CSS is invoked by adding "style" tag inside "head" tag or we can apply the style property 
						for individual elements, but defining "style" of elements inside the head tag is most preferable.
